# Angular Fundamentals

My first Angular app - learning by practicing Angular concepts.

## About This Project

A hands-on learning project where I'm building apps to master Angular fundamentals. Each component is a practice exercise exploring key concepts like components, data binding, TypeScript, and more.

## Tech Stack

- **Angular 21** - Frontend framework
- **TypeScript** - Typed JavaScript
- **Tailwind CSS** - Styling
- **pnpm** - Package manager

## Getting Started

### Prerequisites
- Node.js 20+
- pnpm (`npm install -g pnpm`)

### Install & Run

```bash
cd angular-fundamentals
pnpm install
pnpm start
```

Open [http://localhost:4200](http://localhost:4200) in your browser.
